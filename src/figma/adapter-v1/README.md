# Figma Adapter V1

## 设计原则

从"视觉正确但结构混乱"的 Figma 中提取"设计师想要的视觉细节"，生成"结构规范的 DSL"。

## 核心流程

```
Figma JSON (混乱结构)
    ↓
1. 提取视觉细节 (尺寸、颜色、圆角、阴影)
2. 识别内容资产 (文字、图片、图标)
3. 过滤装饰节点 (白色背景、小装饰条)
4. 识别结构模式 (3x3 grid、flex 列表)
    ↓
规范的 DSL (结构清晰 + 视觉细节)
    ↓
AI 修正 (像素级微调)
    ↓
最终 HTML
```

## 与 AI 的分工

| Adapter V1 | AI |
|------------|-----|
| 提取精确的视觉属性 | 像素级微调 |
| 生成规范的结构 | 语义化优化 |
| 过滤明显的装饰 | 最终验证 |
