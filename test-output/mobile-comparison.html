组件映射对比

同一功能在 Web 和 Mobile 的组件选择差异

导航组件

    Web: Sidebar 侧边栏

    Mobile: BottomTabs 或 Drawer

数据展示

    Web: Table 表格 (5+ 列)

    Mobile: List + Card (纵向)

弹窗组件

    Web: Modal 居中弹窗

    Mobile: Sheet 底部弹层

选择组件

    Web: Select 下拉选择

    Mobile: ActionSheet 操作菜单

切换组件

    Web: Tabs 选项卡

    Mobile: Segment 分段器
